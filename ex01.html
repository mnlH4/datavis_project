Requete sur wikidata

Theme : Scientifique

Code de la requete : 

            
                SELECT ?scientifique ?prénom ?nom_de_famille ?nom_de_familleLabel ?date_de_naissance ?lieu_de_naissance ?lieu_de_naissanceLabel ?occupation ?occupationLabel ?domaine_d_activité ?domaine_d_activitéLabel ?scolarité WHERE {
                    SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                    ?scientifique wdt:P106 wd:Q901.
                    OPTIONAL { ?scientifique wdt:P735 ?prénom. }
                    OPTIONAL { ?scientifique wdt:P734 ?nom_de_famille. }
                    OPTIONAL { ?scientifique wdt:P569 ?date_de_naissance. }
                    OPTIONAL { ?scientifique wdt:P19 ?lieu_de_naissance. }
                    OPTIONAL { ?scientifique wdt:P106 ?occupation. }
                    OPTIONAL { ?scientifique wdt:P101 ?domaine_d_activité. }
                    OPTIONAL { ?scientifique wdt:P69 ?scolarité. }
                  }
                  LIMIT 2000
